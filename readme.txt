=== External Files Optimizer ===
Contributors: julienappert
Tags:optimize, optimizer, speed,external, file, plugin, css, js
Requires at leat: 2.7
Tested up to:2.9.1
Stable tag:0.1.2

Description: automatically combine and compress css/js files generate with wp_head() and wp_footer()

== Installation ==

1. Upload external-files-optimizer directory to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it !
